• Piept die Kamera (Di-Di-Di)? Wenn nicht, setzen Sie die Kamera zurück und vergewissern Sie sich, dass Sie das Piepen hören. Versuchen Sie es anschließend erneut über AirLink.

  • Sie haben es dreimal über AirLink versucht, aber keinen „Dong“ gehört? Bitte versuchen Sie die alternativen Methoden:

    Smart Scannen (HiKam S5, Q7, A7 2016 Version)

    AP-Modus (HiKam S6, HiKam A7 2018 Version)

  • Manche Router bieten gleichzeitig ein 2,4Ghz-WLAN und ein 5Ghz-WLAN. Prüfen Sie im Router, dass Ihr Smartphone mit dem 2,4Ghz-WLAN verbunden ist. Im Zweifelsfall deaktivieren Sie vorübergehend das 5Ghz-WLAN, Sie können es wieder aktivieren, sobald AirLink abgeschlossen wurde.

  • Trennen Sie den WLAN-Router eine Minute lang vom Strom. Verbinden Sie ihn wieder und warten Sie ab, bis er mit dem Internet verbunden ist. Versuchen Sie es erneut über AirLink.

  • Vergewissern Sie sich, dass Ihr WLAN-Router folgende Anforderungen erfüllt:
    • MAC-Filter ist deaktiviert (oder wird vorübergehend deaktiviert, bis AirLink abgeschlossen wurde)
    • DHCP ist aktiviert
    • WLAN-Verschlüsselung: WPA2-PSK oder WPA2-AES
    • Deaktivieren Sie Optionen am WLAN-Router: „Gastzugang“ und „Nur Surfen und Mailen“
    • Ändern Sie den WLAN-Kanal zu „Automatisch“
